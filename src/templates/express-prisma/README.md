# Generator for easier and faster backend dev

There is express for making you express your feelings more. Theres also typescript, prisma and others ;)

## How to use

* At the beggining of everything (if you didnt use [hajabusa_tg](https://github.com/daichi-ranguru/hajabusa_template_generator)): `npm init && yarn install`
* To init prisma: `npx prisma init`
* To run dev: `yarn dev`
* To run prod: `yarn prod`
